## 🚀 Deployment Guide

### Quick Start (Netlify - Recommended)
```bash
# Install Netlify CLI
npm install -g netlify-cli

# Deploy
cd ~/Code/energized-seo-tool/site
netlify deploy --prod
```

### Alternative Options
- **Vercel**: `vercel deploy`
- **Cloudflare Pages**: Via dashboard or Wrangler CLI
- **Traditional Hosting**: Upload via FTP/SFTP

### Domain Setup
1. Connect energizedseo.com
2. Enable HTTPS (automatic on most platforms)
3. Set up redirects (www ↔ non-www)
4. Update DNS records
